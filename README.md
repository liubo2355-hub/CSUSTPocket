掌上长理是一款面向长沙理工大学师生的轻量化校园工具 App，提供课表、成绩、考试安排、网络课程、宿舍电量和校园服务等功能。

## 主要功能

### 功能页面

- 绑定教务系统后，可使用教务系统相关功能，例如查询课表，成绩或考试安排
- 在不登录的情况下，也可以进行空闲教室，电费查询等功能

## FAQ

### 在掌上长理中输入自己的学生账号与密码是否有泄露的风险

学号和密码仅用于访问相应校园系统，敏感凭据保存在本地。请妥善保管设备与账号，并以学校官方渠道发布的信息为准。
