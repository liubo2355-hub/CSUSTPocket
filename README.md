# 掌上长理

便捷、清晰的校园信息工具。

掌上长理是一款面向长沙理工大学师生的校园工具应用，提供课表、成绩、考试安排、网络课程、宿舍电量、校园服务等功能，帮助用户更便捷地获取与管理校园信息。

## 主要功能

- **课表查询**：查看本周课表，支持自定义课程与长按拖动，离线也能查看。
- **成绩查询**：从教务系统获取成绩，展示学分、绩点与课程详情，数据仅保存在本地。
- **考试安排**：展示今日及近期考试，优先提示最近一场。
- **网络课程中心**：查看所有网络课程与待提交作业。
- **宿舍电量**：查询宿舍电费余额与用电趋势，支持绑定宿舍。
- **校园服务**：空教室查询、校园地图、校历、电费充值、WebVPN、评教系统等工具。
- **桌面小组件**：支持课表、电费、考试、成绩等信息的桌面快捷展示。

## 常见问题

### 在掌上长理中输入学号密码是否有泄露风险？

学号和密码仅用于访问学校相应业务系统，敏感凭据保存在本地，不会上传至云端服务器。请妥善保管设备与账号，并以学校官方渠道发布的信息为准。

### 如何加入开发？

请通过本仓库提交 Pull Request 参与开发。
